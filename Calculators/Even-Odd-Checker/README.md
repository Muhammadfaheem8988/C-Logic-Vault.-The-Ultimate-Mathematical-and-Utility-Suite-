This program is a fundamental logic tool that classifies any given integer as either Even or Odd. It serves as a primary example of using conditional branching based on arithmetic remainders.
The program logic follows this mathematical rule:
Even Numbers: Any integer that is perfectly divisible by 2 (n (mod 2) = 0).
Odd Numbers: Any integer that leaves a remainder of 1 when divided by 2 (n (mod 2) != 0).
The implementation uses the modulus operator (%) to perform this check instantly, providing a quick and accurate classification for any positive or negative integer.

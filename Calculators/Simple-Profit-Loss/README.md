This program is a fundamental financial tool used to determine whether a transaction resulted in a profit or a loss. 
By comparing the cost price (initial investment) against the selling price (revenue), the program provides clear feedback on the financial performance of an item.
The program logic follows these core principles:
Pre-calculation: It calculates both potential profit (SP - CP) and potential loss (CP - SP) simultaneously.
Conditional Branching: It uses an if-else structure to check the relationship between prices:
Profit: Triggered when Selling Price > Cost Price.
Loss: Triggered when Cost Price > Selling Price.
Output: Displays the specific amount gained or lost based on the result of the comparison.

This program identifies the largest of three integers using a nested conditional structure. Unlike a sequential comparison, this approach uses a decision tree to eliminate smaller values efficiently.

The logic follows a branching path:

Primary Branch: Compares the first and second numbers.

Secondary Branch (If A > B): Compares the first number with the third.

Secondary Branch (If B > A): Compares the second number with the third.

Result: Directly prints the winning value from the final comparison.

This is a Robust GCD and LCM Calculator designed to handle the full range of integers, including negative values. It implements a pre-processing step to normalize inputs, ensuring that the mathematical properties of divisors and multiples remain consistent.
Technical Improvements:Absolute Value Normalization: Automatically converts negative inputs to positive integers before processing.
Zero-Guard Logic: Prevents division-by-zero errors in the LCM calculation if the user inputs zero values.
Mathematical Accuracy: Adheres to the standard definition that GCD(a, b) is always non-negative.

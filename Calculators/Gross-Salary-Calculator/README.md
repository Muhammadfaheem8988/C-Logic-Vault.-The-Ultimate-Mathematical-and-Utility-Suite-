This program calculates the total Gross Salary of an employee based on their basic pay and fixed percentage-based allowances. It is a common utility for HR and payroll logic.
The calculation follows these steps:
DA (40%): Calculated as 0.4 * Basic Salary.
HRA (20%): Calculated as 0.2 * Basic Salary.
Total Gross: The final sum of Basic + DA + HRA.

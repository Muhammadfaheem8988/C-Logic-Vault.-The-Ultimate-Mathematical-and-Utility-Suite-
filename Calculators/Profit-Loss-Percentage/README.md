This program calculates the absolute financial gain or loss and expresses that value as a percentage relative to the original investment (Cost Price). This is a critical metric for understanding the "margin" of a sale.
The mathematical logic used is:
Profit Percentage: (Profit/Cost_Price)*100.
Loss Percentage: (Loss/Cost_Price)*100.
The program ensures that if the cost price and selling price are equal, it identifies the "Break-even" point where no percentage gain or loss exists.

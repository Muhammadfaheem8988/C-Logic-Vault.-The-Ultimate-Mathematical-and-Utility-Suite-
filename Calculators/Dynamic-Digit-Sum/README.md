This program is an advanced version of the digit-sum tool. Instead of being hard-coded for 4-digit numbers, it utilizes a Dynamic Reduction Algorithm to isolate the boundary digits of an integer of any length.The logic follows a two-stage process:Direct Extraction: The last_digit is captured instantly using the modulus operator (% 10).Iterative Reduction: A while loop repeatedly divides the number by 10. This effectively "peels off" the rightmost digits one by one until only the leading (first) digit remains.

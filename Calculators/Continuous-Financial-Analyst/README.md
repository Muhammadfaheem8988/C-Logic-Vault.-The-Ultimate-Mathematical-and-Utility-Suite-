This program is a persistent financial analysis tool designed for continuous calculation of business margins. It automates the discovery of cost prices and profit/loss percentages, allowing users to evaluate multiple inventory batches without restarting the software.

The program utilizes a controlled loop structure to manage:

Dynamic Financial Tracking: Calculates margins as a percentage of the initial investment (Cost Price).

Unit Economics: Identifies the "Price Floor" (Cost Price per Item) required to avoid future losses.

Error Handling: Includes checks to prevent mathematical errors like division by zero when entering item counts.

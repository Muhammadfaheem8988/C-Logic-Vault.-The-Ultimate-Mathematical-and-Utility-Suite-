This program provides a dynamic solution for calculating the sum of digits for any integer. By replacing manual division steps with a while loop, the code becomes much more flexible while maintaining the core logic of base-10 decomposition.

The logic follows your established pattern:

The Remainder (% 10): Isolate the rightmost digit to add it to the running total.

The Reduction (/ 10): Discard the processed digit to move to the next place value.

The Loop: Continues automatically until every digit has been processed and added to the sum.

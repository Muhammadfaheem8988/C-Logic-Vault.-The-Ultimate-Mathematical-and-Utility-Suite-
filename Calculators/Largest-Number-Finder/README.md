This program identifies the largest value among three user-provided integers. 
It uses a step-by-step conditional approach to narrow down the maximum value, first comparing a pair of numbers and then checking the result against the final input.
The logic flow is:
1. Initial Comparison: Compare A and B, storing the higher value in a temporary variable.
2. Final Comparison: Compare that temporary variable with C.
3. Result: Output the definitive largest number.

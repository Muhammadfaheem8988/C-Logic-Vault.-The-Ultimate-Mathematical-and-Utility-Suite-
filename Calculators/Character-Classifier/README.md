This program classifies any keyboard input into four distinct categories: Uppercase, Lowercase, Digits, or Special Characters. It relies on the ASCII (American Standard Code for Information Interchange) table to identify characters based on their underlying numeric values.

The Categorization Logic:

Uppercase [65-90]: Represents 'A' through 'Z'.

Lowercase [97-122]: Represents 'a' through 'z'.

Digits [48-57]: Represents numbers '0' through '9'.

Special Characters: Includes symbols, punctuation, and control characters found in the gaps between the letter and digit ranges.

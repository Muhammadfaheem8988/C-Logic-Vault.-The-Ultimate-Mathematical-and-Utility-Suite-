This program is an academic utility designed to aggregate student performance across five subjects. 
It computes the total score and provides a percentage representation, which is a standard requirement for grade reporting.
The program logic follows these steps:
Aggregation: Sums the scores of five distinct subjects into a total_obtained variable.
Normalization: Converts the raw score into a percentage using the formula:
	Percentage = (Obtained_Marks/Total_Marks) * 100
Precision: Uses formatted output to ensure marks and percentages are readable and accurate.

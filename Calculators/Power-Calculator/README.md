This program calculates the result of a number raised to a specific power ($n^p$). It demonstrates the use of an accumulator variable within a loop to perform repeated multiplication.

Technical Breakdown:
	Initialization: The result variable r is initialized to 1 because 1 is the multiplicative identity.
	The Loop: The for loop runs exactly p times. In each iteration, the current result is multiplied by the base number n.
	Efficiency: This provides an $O(p)$ linear time complexity for calculating integer powers.

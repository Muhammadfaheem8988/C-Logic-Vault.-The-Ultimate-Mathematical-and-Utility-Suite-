This program is an accounting utility that evaluates financial performance based on sales data. It determines the initial cost price and unit cost while categorizing the transaction as a profit, loss, or break-even event.

The program logic follows these core accounting principles:
Cost Discovery: Identifies the total investment by relating the selling price to the reported gain or loss.
Conditional Analysis: Uses if-else branching to compare SP and CP.
Profit: If SP > CP.Loss: If CP > SP.
Unit Economics: Breaks down the total cost into an "item-level" cost for inventory tracking.

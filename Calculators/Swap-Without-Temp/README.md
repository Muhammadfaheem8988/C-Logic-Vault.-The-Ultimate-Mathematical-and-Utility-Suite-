This program demonstrates how to swap two variables using Arithmetic Operations. This method is often used to demonstrate logical efficiency and a deep understanding of how values can be manipulated mathematically.
How the math works:
Let's say C=10 and D=20:
Addition: C = 10 + 20 = 30
Subtraction for D: D = 30 - 20 = 10 (D now has C's old value)
Subtraction for C: C = 30 - 10 = 20 (C now has D's old value)
Note: While clever, this method can sometimes lead to "Integer Overflow" if the numbers are extremely large. In those cases, the temporary variable method we did previously is safer!

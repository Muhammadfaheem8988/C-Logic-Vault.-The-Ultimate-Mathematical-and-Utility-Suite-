This program provides a complete mathematical analysis of two integers by finding both their Greatest Common Divisor (GCD) and Least Common Multiple (LCM).

The mathematical logic follows two main steps:

GCD via Euclidean Algorithm: Uses repetitive division to find the largest common factor.

LCM via Product Property: Utilizes the theorem that the product of two numbers is equal to the product of their GCD and LCM.

By storing the original input values before the GCD loop starts, the program can accurately compute the LCM without requiring a separate, less efficient loop.

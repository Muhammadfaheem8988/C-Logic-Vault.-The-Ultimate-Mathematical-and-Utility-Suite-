This program is a continuous, menu-driven geometric calculator that supports repeated operations without requiring a program restart. It features a control loop that remains active until the user selects the exit option.The program provides specific calculation modules for:Rectangles: Computes the area and perimeter from user-defined length and width.Circles: Computes the area and circumference using a fixed value of $3.1415$ for $\pi$.All numerical outputs are formatted to two decimal places, and the interface includes error handling for invalid menu selections.

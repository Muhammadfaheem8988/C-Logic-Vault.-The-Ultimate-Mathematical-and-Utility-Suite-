This program is a menu-driven geometric tool that allows users to select between two different shapes: a rectangle or a circle. Based on the user's selection, the program prompts for the necessary dimensions (length/breadth or radius) and calculates specific properties.For Rectangles: It calculates the area and the perimeter.For Circles: It calculates the area and the circumference using the constant $3.1415$ for $\pi$.The results are displayed with two-decimal precision for accuracy.

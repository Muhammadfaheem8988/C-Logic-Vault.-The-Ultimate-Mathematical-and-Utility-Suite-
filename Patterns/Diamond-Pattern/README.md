This program generates a symmetrical Diamond Pattern. It demonstrates the use of multiple loop blocks to create complex geometry by combining an upright triangle with an inverted one.

Logic Breakdown:

    Upper Section: The first set of loops builds the triangle from 1 star to 5 stars, using decreasing spaces to maintain alignment.

    Lower Section: The second set of loops starts from 4 stars and counts down to 1. This creates the tapering bottom of the diamond.

    Symmetry: By using a consistent space-calculation formula (5 - i) in both sections, the vertical axis of the diamond remains perfectly centered.

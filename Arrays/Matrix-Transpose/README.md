This program performs a Matrix Transpose, which is the process of swapping the row and column indices of every element in a matrix.

The Logic:
	Dimensional Swap: An original matrix of dimensions $M \times N$ results in a transposed matrix of $N \times M$.
	Indexing: The element located at matrix[i][j] is moved to transpose[j][i].
	Visualization: This operation essentially reflects the matrix across its main diagonal (the line from top-left to bottom-right).

This program demonstrates the fundamental use of 2D Arrays (Matrices) in C. It provides a blueprint for handling tabular data, where information is organized into rows and columns.

Technical Breakdown:

Memory Structure: The array matrix[3][4] allocates space for 12 integers arranged in 3 rows and 4 columns.

Nested Loop Traversal: * The Outer Loop manages the row index.

The Inner Loop manages the column index for the current row.

Data Retrieval: Uses the syntax array[i][j] to point to the specific memory address of an element.

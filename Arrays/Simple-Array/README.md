This program demonstrates basic 1D Array manipulation, specifically focusing on sequential data entry and retrieval. It highlights the importance of index management in C programming.
Key Concepts:
Zero-Based Indexing: In C, arrays start at index 0. 
An array of size N has valid indices from 0 to N-1.
Sequential Access: Using a for loop to traverse memory addresses in a linear fashion.
Buffer Safety: Ensuring loop boundaries do not exceed the declared array size to prevent memory corruption or "segmentation faults."

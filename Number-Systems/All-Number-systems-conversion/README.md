This C program performs various number system conversions between Decimal, Binary, and Octal formats. It utilizes mathematical power functions to calculate positional values and provides a step-by-step conversion for:

Octal to Decimal: Converts base-8 input to base-10.

Decimal to Octal: Converts base-10 input to base-8.

Binary to Decimal: Converts base-2 input to base-10.

Decimal to Binary: Converts base-10 input to base-2.

The program is designed to handle long integer inputs for larger conversion values.

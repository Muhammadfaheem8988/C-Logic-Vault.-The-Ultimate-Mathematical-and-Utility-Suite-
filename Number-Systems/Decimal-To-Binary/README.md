This program converts a base-10 (decimal) integer into its base-2 (binary) equivalent. It uses a mathematical approach involving powers of 10 to reconstruct the binary digits into a single long int variable.

The conversion process follows these steps:

1. Takes a decimal integer as input.

2. Uses a while loop to repeatedly find the remainder when the number is divided by 2.

3. Multiplies each remainder by increasing powers of 10 to position it correctly in the final output.

4. Displays the completed binary representation to the user.

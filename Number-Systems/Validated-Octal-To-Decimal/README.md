This program converts an octal (base-8) number into its decimal (base-10) equivalent while performing input validation. It ensures that the user-provided number contains only valid octal digits (0–7) before proceeding with the mathematical conversion.The program features:Input Validation: As each digit is isolated using the modulus operator, the program verifies it does not exceed 7. If an invalid digit (8 or 9) is found, the process stops and issues a warning.Base-8 Conversion: For valid inputs, it calculates the decimal value by multiplying each digit by $8^n$, where $n$ represents the digit's position.Accurate Results: It uses long int data types to support larger numeric conversions and provides a clear summary of the transformation.

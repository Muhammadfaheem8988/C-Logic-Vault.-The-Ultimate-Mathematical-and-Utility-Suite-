This program converts a decimal (base-10) integer into its octal (base-8) equivalent. It utilizes a mathematical algorithm that extracts remainders through successive division and reconstructs them into an octal format using powers of ten to maintain digit positioning.The conversion process involves:Successive Division: The input decimal number is repeatedly divided by 8.Remainder Extraction: The remainder of each division (ranging from 0 to 7) represents an octal digit.Positional Reconstruction: Each remainder is multiplied by an increasing power of 10 ($1, 10, 100, \dots$) to build the final octal value as a single long integer.Output: The program displays the original decimal value alongside its calculated octal counterpart.

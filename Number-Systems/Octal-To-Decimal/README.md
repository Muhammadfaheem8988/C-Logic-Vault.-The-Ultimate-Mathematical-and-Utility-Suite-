This program converts an octal (base-8) number into its corresponding decimal (base-10) integer. The conversion is achieved by isolating each digit of the octal input and multiplying it by the appropriate power of 8 based on its position.The program logic follows these steps:Digit Extraction: Uses the modulus operator (% 10) to pull individual octal digits (0 through 7) from the input.Positional Weighting: Uses the pow(8, n) function to calculate the value of each digit's position (e.g., $1, 8, 64, 512, \dots$).Summation: Accumulates the products of the octal digits and their respective weights into a final decimal total.Output: Displays the original octal value and its converted decimal result.

This program converts a binary number (base-2) into a decimal number (base-10) while including a data validation layer. It ensures that the input consists strictly of binary digits (0 and 1) before performing any mathematical conversions.The program logic includes:Input Validation: As each digit is extracted using the modulus operator, the program checks if the value exceeds 1. If an invalid digit is found, the process terminates with an error message.Positional Calculation: For valid inputs, each bit is multiplied by $2^n$ (where $n$ is the position index) and added to a running total.Result Reporting: Displays the final decimal integer only if the entire input string passes the binary criteria.

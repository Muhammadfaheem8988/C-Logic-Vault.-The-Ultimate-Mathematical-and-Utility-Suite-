This program converts a binary (base-2) number into its corresponding decimal (base-10) integer. It processes the input by isolating each binary digit and multiplying it by the appropriate power of 2 based on its position.The program logic follows these steps:Digit Extraction: Uses the modulus operator (% 10) to pull individual bits (0 or 1) from the input.Positional Weighting: Uses the pow(2, n) function to calculate the value of each bit's position (e.g., $1, 2, 4, 8, \dots$).Summation: Accumulates the products of the bits and their weights into a final decimal total.Output: Displays the converted decimal integer.

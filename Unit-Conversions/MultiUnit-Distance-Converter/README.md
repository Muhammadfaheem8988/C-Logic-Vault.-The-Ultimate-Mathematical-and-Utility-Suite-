This program is a flexible distance conversion utility that allows users to input a value in their preferred unit (Kilometers, Meters, or Feet) and see the equivalent measurements in four other units. It uses a conditional menu structure to handle different starting points and applies consistent conversion factors to bridge the metric and imperial systems.The conversion logic includes:Metric Conversions: Handles the relationship between Kilometers and Meters (factor of $1000$).Bridge Conversions: Connects Meters to Feet (factor of $3.28$).Imperial Conversions: Handles the relationship between Feet, Inches ($12$), and Centimeters ($2.5$).The program ensures that regardless of the starting unit, a full breakdown of the distance is provided in a clean, formatted output.

This program performs a series of unit conversions for geographic distances. Starting from an input in kilometers, it sequentially calculates the equivalent distance in meters, feet, inches, and centimeters using standard conversion constants.The program logic follows these specific conversion steps:Kilometers to Meters: Multiplies by $1000$.Meters to Feet: Multiplies by $3.28$.Feet to Inches: Multiplies by $12$.Inches to Centimeters: Multiplies by $2.5$.The results are displayed in a formatted list, providing a comprehensive breakdown of the distance across both metric and imperial systems.

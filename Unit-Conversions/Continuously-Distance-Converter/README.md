This program is a robust, interactive unit conversion tool that handles distance measurements across several metric and imperial units. It uses a persistent loop and a selection menu to allow the user to input data in Kilometers, Meters, or Feet and receive a full conversion breakdown.The program utilizes the following core conversion logic:Metric System: Kilometers to Meters (factor of $1000$).Imperial System: Feet to Inches (factor of $12$) and Inches to Centimeters (factor of $2.5$).Cross-System: Meters to Feet (factor of $3.28$).The application includes a clean interface with an exit condition and input validation to ensure a smooth user experience.
